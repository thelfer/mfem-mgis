Enumerator
CONSTANTSTIFFNESS 

the initial prediction matrix or the initial tangent coherent matrix or the initial secant matrix (normally equal to the elastic stiffness) is used for all iterations

CONSTANTSTIFFNESSBYPERIOD 

the prediction matrix or the first tangent coherent matrix or the first secant matrix is used for all iterations and updated and factorized at at periods

UPDATEDSTIFFNESSMATRIX 

the tangent coherent matrix or the secant matrix is computed and factorized at each iteration
