constexprnoexcept

a portable implementation of the std::fpclassify function based on the MUSL library (https://www.musl-libc.org)

Returns
the class of the floatting point value according to the IEEE754 standard. Possible return values are:
  • FP_NORMAL: standard floatting point value
  • FP_SUBNORMAL: subnormal floatting point value
  • FP_NAN: NaN value
  • FP_INFINITE: +Inf or -Inf value
