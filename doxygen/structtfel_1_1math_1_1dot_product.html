Detailed Description

template<unsigned int N>
class tfel::math::dotProduct< N >

dotProduct calculates a generalized inner product of the ranges [first1,first1+N) and [first2,first2+N)

Parameters
Nnumber of element of each ranges

Contrary to inner_product, dotProduct takes automatically into account the fact that complex must be conjugated

The first version of dotProduct returns init plus the inner product of the two ranges [1]. That is, it first initializes the result to init and then computes:

\[\displaystyle\mathrm{init}+\sum_{i=0}^{N-1}\overline{(*(p+i))}*(*(q+i))\]

Parameters

InputIterator2iterator to the second range

Precondition
  • InputIterator1 is a model of Input Iterator.
  • InputIterator2 is a model of Input Iterator.
  • T is a model of Assignable.
  • If x is an object of type T, y is an object of InputIterator1's value type, and z is an object of InputIterator2's value type, then x + y * z is defined.
  • The type of x + y * z is convertible to T.
