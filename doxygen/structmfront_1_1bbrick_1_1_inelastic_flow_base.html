Member Enumeration Documentation

◆ PorosityEffectOnFlowRule

enum mfront::bbrick::InelasticFlowBase::PorosityEffectOnFlowRule
protected

policy on how the porosity affects the flow rule

Enumerator
UNDEFINED_POROSITY_EFFECT_ON_EQUIVALENT_PLASTIC_STRAIN 

default choice. The effect of the porosity depends on the stress criterion (see the StressCriterion::getPorosityEffectOnEquivalentPlasticStrain method).

NO_POROSITY_EFFECT_ON_EQUIVALENT_PLASTIC_STRAIN 

This value indicate that the flow rule is not affected by the porosity evolution.

STANDARD_POROSITY_CORRECTION_ON_EQUIVALENT_PLASTIC_STRAIN 

This value indicate that the flow rule must be corrected by the standard \((1-f)\) factor where \(f\) is the porosity.

◆ cosine_threshold

double mfront::bbrick::InelasticFlowBase::cosine_threshold = 2
protected

minimum value of the cosine of the angle between two successive estimates of the flow direction. If the computed angle is lower than this threshold, the Newton step is rejected.

Note
the value of this parameter must be in the range [-1:1]. If this parameter is not in that range (which is the default), the code checking the value of the cosine is not generated.
