Detailed Description

class Vect3 handles a vector of double in 3 dimensions

The first author of this method is A. Etcheverry who basically got rid of all the valarray<double>(3) in NUMODIS to improve computation efficiency.

This class offers several interesting operators to make 3D calculations almost as simple as in Fortran90 ;-)
