This function computes the RambergOsgood paramters K and n which allow the approximation of the traction curve by the relation: [ p = K(s/E)**n ] where:

  • p is the cumulated plastic strain
  • s is the von mises stress
  • E is the Young modulus

This functions can be used with the mfront' IsotropicPlasticMisesFlow parser like this:

@LocalVariable real K_p;
@LocalVariable real n_p;
@InitLocalVariables{
using namespace tfel::material;
const real Rp02=...;
const real Rm=...;
const real Ag=...;
real K_r;
real n_r;
computeRambergOsgoodParameters(K_r,n_r,young,Rp02,Rm,Ag);
K_p=1./K_r;
n_p=1./n_r;
}
@FlowRule{
const real p0 = 1.e-6;
const real tmp = young*pow(K_p*(p+p0),n_p-1);
f = seq-tmp*K_p*(p+p0);
df_dseq = 1.;
df_dp = -n_p*K_p*tmp;
}
