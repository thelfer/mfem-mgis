template<tfel::math::ScalarConcept T>
TFEL_HOST_DEVICE constexpr std::pair< T, T > tfel::material::computeKappaMu ( const tfel::math::st2tost2< 3u, T > &  A)
constexpr

This function makes the projection of a st2tost2 on the subset of isotropic fourth-order tensors. It uses the quadruple contraction of \(C_i\) with \(J\) and \(K\). It returns the moduli \(k\) and \(mu\). The implementation goes for dimension 3 only.

This function permits to know if a st2tost2 is Isotropic It first makes the projection on the subset of isotropic fourth-order tensors using computeKappaMu and then compare the projected tensor with the original \(A_i\). The precision can be given by the user. The implementation goes for dimension 3 only.
