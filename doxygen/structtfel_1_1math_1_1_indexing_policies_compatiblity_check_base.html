Detailed Description

template<typename IndexingPolicy1, typename IndexingPolicy2>
struct tfel::math::IndexingPoliciesCompatiblityCheckBase< IndexingPolicy1, IndexingPolicy2 >

a base class to compare indexing policies which provide default implementations.

Template Parameters
IndexingPolicy1first indexing policy
IndexingPolicy2second indexing policy
Note
the default implementations are very conservative. For example, the memory layout are reported to be different by default.

staticconstexpr
Returns
if the indexing policies have the same memory layouts.

Two indexing policies can be compatible (at compile-time and at runtime), i.e. one may transfer data from one indexing policy to another, but the memory layout of the data may differ. For example, one may assign a matrix stored in column-major format to a matrix having a row-major format.
