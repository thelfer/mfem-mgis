Detailed Description

template<unsigned short int N, tfel::math::ScalarConcept StressType>
requires (tfel::math::checkUnitCompatibility< tfel::math::unit::Stress, StressType>())
struct tfel::material::homogenization::elasticity::InclusionDistribution< N, StressType >

This struct represents a distribution of inclusions as an abstract class, child of class Phase. It contains an inclusion and a pure virtual method computeMeanLocalisator which returns the mean localisator of the distribution of inclusions, for a given matrix elasticity in parameter.
