This function computes the implicit flow stress \(\sigma^{\star}\) of the Gurson-Tvergaard-Needleman model.

\(\sigma^{\star}\) is implicitely defined by the following relation:

\[ \left(\frac{\sigma_{\mathrm{eq}}}{\sigma^{\star}}\right)^{2}+ 2\,q_{1}\,f^{\star}\,\cosh\left(\frac{q_{2}\,\mathrm{tr}\left(\underline{\sigma}\right)}{2\,\sigma^{\star}}\right) -1-q_{1}\,f^{\star}=0 \]

Returns
the value of \(\sigma^{\star}\)
Parameters
[in]seq: Von Mises stress
[in]skk: stress trace
[in]fst: void volume fraction
[in]q1: \(q_1\) parameter
[in]q2: \(q_2\) parameter
