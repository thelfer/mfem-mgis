template<unsigned short N, typename ValueType>
class tfel::math::stensor< N, ValueType >

Template Parameters
Nthe spatial dimension, see StensorDimeToSize for details.
Tnumerical type used, by default, double
Precondition
This class is only defined for N=1u,2u and 3u.
See also
StensorDimeToSize and StensorSizeToDime.
