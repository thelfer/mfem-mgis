Detailed Description

template<unsigned short N, typename ScalarType = double>
struct tfel::math::TinyRungeKuttaSolverBase< N, ScalarType >

a base class for a system of ordinary differential equations based on explicit methods of the Runge-Kutta family where the size of the solution is known at compile-time.
