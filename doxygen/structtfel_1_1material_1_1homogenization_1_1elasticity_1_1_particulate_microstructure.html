Detailed Description

template<unsigned short int N, tfel::math::ScalarConcept StressType>
requires (tfel::math::checkUnitCompatibility< tfel::math::unit::Stress, StressType>())
struct tfel::material::homogenization::elasticity::ParticulateMicrostructure< N, StressType >

This struct defines a matrix-inclusion microstructure (or particulate microstructure) as a child of Microstructure. This type of microstructure is viewed as an array of InclusionDistribution, with a matrixPhase.
