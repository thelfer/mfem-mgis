write an mfm-test-generator test to a file.

The file format is deduced from the extension of the file.

Currently, only extensions associated with the madnex file format (https://github.com/thelfer/madnex) are supported if TFEL is compiled with support of this file format. Those extensions are: mdnx, madnex (deprecated) or edf (experimental data file, deprecated). Note that the behaviour member of the metadata must be specified for export in the madnex file format.
