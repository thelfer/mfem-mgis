◆ elastic_material_properties_epts

std::vector<std::string> tfel::system::ExternalBehaviourDescription::elastic_material_properties_epts

names of entry points associated with elastic material properties exported by the behaviour, if any. This is currently only supported by a few interfaces (Cyrano, Cast3M).

The size of this member is either:

  • 0: which means that the behaviour does not export its elastic properties.
  • 2, which means that the elastic behaviour is isotropic.
  • 9, which means that the elastic behaviour is orthotropic.
