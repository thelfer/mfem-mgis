Detailed Description

template<typename IndexingPolicy1, typename IndexingPolicy2, typename IndexingPolicy2::size_type Stride = getUnderlyingArrayMinimalSize<IndexingPolicy2>()>
struct tfel::math::FixedSizeIndexingPoliciesCartesianProduct< IndexingPolicy1, IndexingPolicy2, Stride >

an indexing policy build as the cartesian product of the two fixed size indexing policies.

The memory position associed with the index (i0, ..., iN, j0, ..., jM) is given by:

\[ p1(i0, ..., iN) * Stride + p2(j0, ..., jM) \]
