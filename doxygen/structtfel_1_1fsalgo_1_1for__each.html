Detailed Description

template<unsigned int N>
class tfel::fsalgo::for_each< N >

For_each applies the function object f to each element in the range [first, first+N); f's return value, if any, is ignored. Applications are performed in forward order, i.e. from first to last. For_each returns the function object after it has been applied to each element.

Parameters
Nnumber of element to which f will be applied

This documentation is mostly inspired from: http://www.sgi.com/tech/stl/for_each.html

for_each.cxx for some elementary tests.
