template<typename NumType>
struct tfel::material::MetallicCFCGenericSlidingSystemsInteractionMatrix< NumType >

Given six interaction coefficients, this class builds only once a generic interaction matrix. This class shall not be called if the interaction coefficients may evolve with time~: the matrix is only compute at the first call.
