Detailed Description

template<FunctionalSpaceConcept Space, size_type N, bool is_mutable = true>
requires (N > 0)
struct mgis::function::StridedCoalescedMemoryAccessFunctionViewBase< Space, N, is_mutable >

a class meant to describe a function, each component of which is stored contiguously in non interleaved manner using the following scheme:

| <------- Component 1 ---------> |....| <----- Component Nc ---------> |
+-------++-------++------++-------+----+-------++------++------++-------+
| Elt 1 || Elt 2 || .... || Elt N |....| Elt 1 ||Elt 2 || .... || Elt N |
+-------++-------++------++-------+----+-------++------++------++-------+
