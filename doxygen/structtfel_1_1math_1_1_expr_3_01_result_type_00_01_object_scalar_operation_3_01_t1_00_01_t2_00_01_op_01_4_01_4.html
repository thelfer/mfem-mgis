Detailed Description

template<typename ResultType, typename T1, typename T2, typename Op>
struct tfel::math::Expr< ResultType, ObjectScalarOperation< T1, T2, Op > >

an Expr object allows the lazy evaluation of a binary operation between a mathematical object and scalar.

Template Parameters
ResultType:
T1: type of the left argument of the binary operation
T2: type of the right argument of the binary operation
Op: the binary operation
