Detailed Description

a simple class used to:

  • handle the life time of the mesh and the finite element collection.
  • create and handle a finite element space

◆ FiniteElementDiscretization() [1/5]

mfem_mgis::FiniteElementDiscretization::FiniteElementDiscretization ( const Parameters &  )

constructor

Parameters
[in]paramsparameters

The following parameters are expected:

  • Parallel (boolean): if true, a parallel computation is to be be performed. This value if assumed to be false by default.
  • MeshFileName (string): mesh file.
  • FiniteElementFamily (string): name of the finite element family to be used. Supported families are:
    • H1: The default value if H1.
  • FiniteElementOrder (int): order of the polynomial approximation.
  • UnknownsSize (int): number of components of the unknows
  • NumberOfUniformRefinements (int): number of uniform refinements applied to the mesh
  • GeneralVerbosityLevel (int): with large positive numbers, expect more verbosity
