Detailed Description

template<bool parallel>
struct mfem_mgis::ParaviewExportIntegrationPointResultsAtNodesImplementation< parallel >

a post-processing to export integration points results to paraview after projecting them to nodes

The functions to be exported can be either:

  1. automatically extracted from the materials defined in a nonlinear evolution problem
  2. explicitely given by the user
