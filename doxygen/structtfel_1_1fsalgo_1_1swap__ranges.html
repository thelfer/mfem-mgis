Detailed Description

template<unsigned int N>
class tfel::fsalgo::swap_ranges< N >

swap_ranges swaps each of the elements in the range [first1, first1+N) with the corresponding element in the range [first2, first2 + N). That is, for each integer n such that 0 <= i < N, it swaps *(first1 + i) and *(first2 + i). The return value is first2 + N.

Parameters
Nsize of the ranges to be swapped.
