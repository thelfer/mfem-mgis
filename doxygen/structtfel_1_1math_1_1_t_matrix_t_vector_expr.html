Detailed Description

template<unsigned short N, unsigned short M, MatrixConcept A, VectorConcept B>
struct tfel::math::TMatrixTVectorExpr< N, M, A, B >

A placeholder for the computation of product of a tiny matrix and a tiny vector.

Template Parameters
N: numbers of rows of the matrix
M: numbers of columns of the matrix
A: type of the tiny matrix object (can be a reference)
B: type of the tiny vector object (can be a reference)
