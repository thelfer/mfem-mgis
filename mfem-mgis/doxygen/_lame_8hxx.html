Detailed Description

This file defines various functions dealing with Lame's coefficients. Using Lame's coefficients, linear elastic behaviour may be written~:

\[ \underline{\sigma} = \lambda*\mathrm{tr}\left(\underline{\epsilon}\right)\underline{I}+2\mu\underline{\epsilon} \]
