static void mfront::SearchPathsHandler::addMadnexSearchPath ( const std::string &  )
static

append a madnex search path

The path has one of the form:

  • <file_path>
  • <file_path>:<material_id>
  • <file_path>:<material_id>:<mkt>

If the material identifier is not specified, all materials are selected, including the empty one.

<material_id> can be a regular expression. <material_id> can also have the special value <none>.

<mkt> can have one of the values MaterialProperties, Behaviours or Models.

static std::string mfront::SearchPathsHandler::search ( const std::string &  )
static

search a file in the specified paths. The file is searched using the following conventions :

  • search in the current directory.
  • search in the search paths from the most recently declared search path to the older ones.

Search paths declared through the MFRONT_INCLUDE_PATH are stored at the end of the list.
