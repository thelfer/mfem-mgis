Detailed Description

template<unsigned int N>
class tfel::fsalgo::equal< N >

Equal returns true if the two ranges [first1, first1+N) and [first2, first2 + N) are identical when compared element-by-element, and otherwise returns false.

Parameters
Nnumber of element to be compared.

This documentation is mostly inspired from: http://www.sgi.com/tech/stl/equal.html

See also
transform.cxx for some elementary tests.
