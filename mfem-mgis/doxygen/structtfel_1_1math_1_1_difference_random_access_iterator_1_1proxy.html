Detailed Description

template<typename iterator_base>
struct tfel::math::DifferenceRandomAccessIterator< iterator_base >::proxy

a proxy class

This proxy is used so that an expression such as &p[o], where p is a DifferenceRandomAccessIterator and o is an offset, returns a DifferenceRandomAccessIterator.

Such expressions are widely used in MFront to initialize variables and we can not change this easily.
